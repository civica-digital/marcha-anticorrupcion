# Antidoping Anticorrupción:

El sistema nacional anticorrupción, aterrizados en la plataforma nacional digital, en si habla de un grafo de contratos, contratistas e instituciones con contratan. Aprovechando esa estructura de información, podemos hacer:

- Una herramienta para recibir denuncias y reconocer de quien se está hablando
- Una estructura para las pruebas anticorrupción, de forma que sirva de antidoping; o de marcha, de la química analitica de las pruebas que se hacen para identificar los reactivo de una solución

En este proyecto
- La integración que refleja la posibilidad de realizar el análisis de los datos como un grafo/sistema experto. (Integrador)
- Diseñamos el proceso de procesamiento de lenguaje natural para poder clasificar denuncias con respecto al texto recibido; Y encontrar de qué contratos, contratistas, instituciones, y personas implicadas. (denuncia)
- Planteamos el proceso de la corrida (Estandar y estructura de abstracción), donde se hacen distintas pruebas para encontrar los distintos tipos de corrupción  http://ley3de3.mx/es/10-tipos-de-corrupcion/ (antidopingº)
- Ejemplos de consumo de ambos productos: Visualización de información y diseño conversacional (Ver presentación)
